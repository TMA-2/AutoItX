# Get-AU3ControlText

## SYNOPSIS

Retrieves the text from a control.

## SYNTAX

### Text (Default)

```
Get-AU3ControlText [-Title] <string> [[-Text] <string>] [[-Control] <string>] [<CommonParameters>]
```

### Handle

```
Get-AU3ControlText [-WinHandle] <IntPtr> [-ControlHandle] <IntPtr> [<CommonParameters>]
```

## ALIASES

This cmdlet has no aliases.

## DESCRIPTION

The Get-AU3ControlText cmdlet retrieves the text content from a control within a window. The control is identified by its ID, ClassNameNN, or text within the specified window. This corresponds to the AutoIt ControlGetText() function.
